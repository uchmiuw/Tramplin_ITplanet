# Tramplin
Карьерная платформа «Трамплин»
Веб-приложение для связи IT-студентов, выпускников, работодателей и карьерных центров вузов.

О проекте:
«Трамплин» позволяет публиковать и искать вакансии, стажировки, менторские программы и мероприятия на интерактивной карте или в виде списка с фильтрами (навыки, зарплата, формат работы).

Роли:
Соискатель – резюме, портфолио, отклики, список контактов, настройки приватности.
Работодатель — управление вакансиями, просмотр откликов, верификация компании.
Куратор — модерация контента, верификация компаний, создание других кураторов.

Ключевые функции:
Карта и лента возможностей, избранное (маркеры другим цветом).
Карточка вакансии с тегами (технологии, уровень, занятость).
При наведении на маркер — модульная карточка.

Авторские решения:
Верификация: корпоративная почта + ИНН (API ФНС).
Модерация: пре-модерация + жалобы.
Теги: стартовый набор + предложение новых.
Нетворкинг: рекомендации друзей на вакансии.

Стек:
БД: Google Firebase
Карты: Leaflet + OpenStreetMap
Языки: HTML+CSS+JavaScript
